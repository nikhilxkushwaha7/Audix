# AUDIX

PREMIUM CLI AUDIO DOWNLOADER

---

## DESCRIPTION

AUDIX IS A FAST, CLEAN, AND PROFESSIONAL COMMAND-LINE AUDIO DOWNLOADER BUILT USING PYTHON AND YT-DLP.  
IT ALLOWS USERS TO SEARCH FOR AUDIO ON YOUTUBE, SELECT FROM MULTIPLE RESULTS, AND DOWNLOAD HIGH-QUALITY MP3 FILES WITH A MINIMAL AND DISTRACTION-FREE CLI EXPERIENCE.

---

## FEATURES

- YOUTUBE SEARCH WITH MULTIPLE SELECTABLE RESULTS  
- PAGINATED SEARCH RESULTS (6 RESULTS PER PAGE)  
- HIGH-QUALITY MP3 AUDIO DOWNLOAD (128 / 192 / 320 KBPS)  
- CONFIGURABLE DOWNLOAD PATH  
- CONFIGURABLE AUDIO QUALITY  
- CROSS-PLATFORM SUPPORT (WINDOWS, LINUX, MACOS)

---

## REQUIREMENTS

- PYTHON 3.9 OR NEWER
- FFMPEG INSTALLED AND AVAILABLE IN SYSTEM PATH  

---

## INSTALLATION

1. CLONE OR DOWNLOAD THE RELEASE PACKAGE
2. INSTALL DEPENDENCIES:

   pip install -r requirements.txt

3. RUN THE TOOL:

   python audix.py

---

## USAGE

1. LAUNCH AUDIX FROM TERMINAL
2. SELECT "DOWNLOAD AUDIO"
3. ENTER SONG NAME
4. CHOOSE FROM SEARCH RESULTS
5. AUDIO WILL BE DOWNLOADED TO CONFIGURED PATH

---

## CONFIGURATION

FROM THE SETUP / CONFIG MENU, USERS CAN:

- CHANGE DOWNLOAD PATH
- CHANGE AUDIO QUALITY
- VIEW CURRENT CONFIGURATION

CONFIG IS STORED LOCALLY IN `conf.json`.
